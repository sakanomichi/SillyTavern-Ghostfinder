Ghostfinder

Navigate between boundary messages - the first visible messages that appear after hidden sections in your chat.

Display the lantern button in the chat interface.
When enabled, clicking the lantern button opens the Boundary Index Panel. When disabled, clicking jumps to the previous boundary. The panel is always accessible via the Extensions menu.
Find the last visible message before each hidden section, instead of the first visible message after each hidden section.
Shows detailed information in the browser console for troubleshooting.
